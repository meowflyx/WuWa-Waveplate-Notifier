## ‚ú® Features

- **Real-time Calculation:** Doesn't just count down seconds. It stores a timestamp and calculates your current Waveplates based on the server regeneration rate (1 WP per 6 minutes).
- **Persistent State:** If the bot restarts (and you add a database), it knows exactly how much you regenerated while it was offline.
- **Smart Notifications:** Pings you exactly when you hit 240/240.
- **Quick Actions:**
  - `I just spent it ALL`: Instantly resets counter to 0 and schedules a reminder for 24 hours later.
  - `Refresh Status`: Shows current count and exact time until full cap.
- **AsyncIO:** Built on `python-telegram-bot` v20+, using `JobQueue` for non-blocking execution.
